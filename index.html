My Projects

Search Engine

Efficiently indexes gigabytes of web pages and can scale with expanding data sizes. Developed an interactive web GUI with Streamlit to dynamically displays query time and search results for each query. Search engine achieves sub-300ms ranked searches by leveraging TF-IDF scores and other heuristics to yield search results similar to Google’s.

Computer Vision Sudoku Solver

Developed a Sudoku Solver featuring backtracking search and an intuitive GUI for enhanced user experience. Implemented image pre-processing with OpenCV to transform user-uploaded Sudoku board images into a format compatible for a neural network to perform accurate predictions. Leveraged PyTorch to train a neural network on the MNIST dataset and achieved a 96% accuracy in predictions. Applied the trained model to extract the numbers from user-uploaded Sudoku board images.

Music Release Tracker

Wrote a program to scrape websites and parse their web page contents using Beautiful Soup to extract the release dates of new songs. Developed a Discord bot that interacts with Discord’s API to present the parsed information in a readable format to the user. Self-hosted the bot on a Docker container within an Ubuntu server to ensure a continuous and reliable access. Deployed the bot across multiple Discord communities totaling over 1,600 members.
