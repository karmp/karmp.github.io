Built entirely in Python, this project combines traditional algorithmic problem-solving with deep learning, providing an interactive and intelligent Sudoku experience.

    Music Release Tracker

    Developed and deployed a Discord bot that tracks and presents new song release dates across multiple Discord communities with a combined user base of over a thousand. Built using Python and Beautiful Soup, the bot scrapes release date information from various music blogs and formats the data into structured responses. Users can interact with the bot through Discord commands to view release dates for current, past, or upcoming months, as long as the information remains available on the source websites. The bot dynamically fetches data upon request rather than relying on scheduled updates, ensuring that users always receive the most up-to-date information. By integrating this tool across multiple Discord servers, I provided music enthusiasts with an easy-to-use resource for discovering new releases, improving engagement within these communities.
